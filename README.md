# Ocean Soundscapes
Distraction-free Background Audio



##### Tech Stack
* HTML5 / CSS3 / Sass / Responsive (phone/tablet/desktop)
* Javascript
* Mocha / Chai Testing
* HTML5 Audio (Chrome/iOS/iPad compatible)
* NodeJS / Grunt

##### Installation
1. ```npm install -g grunt-cli``` (if you don't have Grunt already)
2. ```npm install```
